# spring-lite

A very opinionated application to build a smallest application

### Requireemnts

In order to build application, these are the requirements:

- Java 19
- Gradle 8+

### Usage

```bash
$> gradle clean assemble
$> java -jar spring-lite-1.0.0.jar
```
